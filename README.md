# react-toolbox-boilerplate

**Clone the project**

`git clone https://github.com/Venugopal46/react-toolbox-boilerplate.git`.

**Check out to project folder**

`cd react-toolbox-boilerplate`.

**Install packages**

`npm install`

**Run the app:**  

 * dev mode: `npm start`
 * production mode: `npm run build`
